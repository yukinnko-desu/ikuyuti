# いくゆち
連打育成ゲーム

概要
道端で見つけた神社から落ちてきた3歳の「ゆっちん」を、連打で育てるクリッカーゲームです。あなたの連打速度によって、ゆっちんの成長が変化します。

遊び方

育成パート: 画面の画像をクリック連打してゆっちんを育てよう

イベントパート:空腹のゆっちんが現れます

結果画面: あなたの連打速度に応じて、ゆっちんが成長します

隠し要素: 結果画面でさらに100回クリックすると...？

成長の種類
連打速度によって、ゆっちんの成長が変化します:

5.00回/秒以上: 努力家ゆっちん

2.00回/秒以下: マイペースゆっちん

2.00〜5.00回/秒: 普通のゆっちん
